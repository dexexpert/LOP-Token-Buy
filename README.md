## Dependencies

- Node
- NPM

## Getting Started

Install package dependencies

```
npm install
```

Run locally

```
npm start
```

Run linting and tests

```
npm run lint
npm test
```

Build for prod

```
npm run build
```

## Add New Token

- Icon: Add logo to src/assets/tokens folder
- Token info: src/constants/tokens.js
